Наши спектрофотометры подходят для всех типов лабораторий занимающихся рутинными анализами:

  • в области экологического контроля
    • природной воды,
    • сточных вод,
    • подземных вод,
    • питьевой воды,
    • морской воды,
    • атмосферного воздуха,
    • промышленных выбросов,
    • воздуха рабочей зоны,
    • почвы,
    • донных отложений,
    • шельфов;
  • в области контроля качества сырья и готовой продукции во всех сферах производства;

Залогом этого служит высокая точность и надежность наших спектрофотометров по сравнению с другими подобными приборами. Внедренные современные технологии упростили работу с прибором. Пользователь тратит меньше времени для выполнения измерений, чем на других приборах.

Наши спектрофотометры могут использоваться для образовательных и исследовательских лабораторий.

Для образовательных идеально подходят недорогие модели такие как В-1100 и УФ-1100. Они отличаются высокой надежностью точностью и низкой ценой. Эти модели могут быть использованы для обучения студентов основам фотометрического метода измерений. Использование же этих приборов в комплекте с ПО MWavw Professional V3 позволяет сканировать образец в необходимом участке спектра длин волн. Такой комплект не только решает задачи количественного анализа, но и позволяет разрабатывать новые методики без значительных затрат на сверхдорогие спектрофотометры.

Для исследовательских лабораторий могут быть использованы Спектрофотометры УФ-1800, УФ-3000, УФ-3100, УФ-3200, УФ-6100. Они обеспечивают точность измерений не хуже чем у большинства американских, европейских и японских спектрофотометров, а цена этих приборов значительно ниже! Низкая цена обеспечивается не экономией на качестве и точности, а низкой торговой наценкой и отсутствием переплаты за громкое имя. Большое количество дополнительного оборудования облегчает исследовательскую работу.

Точность спектрофотометров обусловлена точностью конструкции оптический схемы и внедренной системой самокалибровки для всех приборов. В основе прибора лежит монохроматор с дифракционной решеткой и система зеркал. Это классическая оптическая схема спектрофотометра для рутинных анализов. Приборы для исследовательской работы часто требуют очень высоких значений точности, но для количественного рутинного анализа такая точность избыточна. Точность Спектрофотометров В-1100, УФ-1100, В-1200, УФ-1200 ТМ ECOVIEW(ЭКОВЬЮ) достаточна для проведения любого анализа по утвержденной методике измерений. Точность Спектрофотометров УФ-1800, УФ-3000, В-3100, УФ-3200, УФ-3100 ТМ ECOVIEW(ЭКОВЬЮ) достаточна для проведения исследовательских работ.

Применение автоматической системы установки длины позволило значительно повысить точность установки у самых недорогих моделей. Использование чувствительных кремневых фотодиодов позволяет поддерживать фотометрическую точность прибора на очень высоком уровне.

Следует отметить, что высокая точность не сильно влияет на цену. Все технологии, используемые в наших спектрофотометрах, отработаны годами. Все новинки внедряются только после всестороннего изучения их влияния на точность прибора. С годами точная электроника становится дешевле для покупателя, и ее цена становится доступной для использования в приборах для рутинных анализов.
